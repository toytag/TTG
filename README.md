# Truth Table Generator

Truth table generator for well-formed-formula consists of propositional variables

## Notation

|Connectors|Standard Notation|In This Program|
|----------|:---------------:|:---------:|
|Negation  |¬                |!          |
|Conjunction|∧               |&          |
|Disconjunction|∨            |&#124;     |
|Implication|→               |->         |
|Anti-implication|←          |<-         |
|Bi-Implication|↔︎            |<->        |

  Example:  
  Standard -> (¬p→q)∨q∧(r↔︎s)  
  In this Program -> (!p->q)|q&(r<->s)
