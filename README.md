# TTG

Truth table generator for well-formed-formula consists of propositional variables

## Notation

`!`     is for Negation         `¬`  
`&`     is for Conjunction      `∧`  
`|`     is for Disconjunction   `∨`
`->`    is for Implication      `⟶`  
`<-`    is for Anti-implication `⟵`  
`<->`   is for Bi-Implication   `⟷`
